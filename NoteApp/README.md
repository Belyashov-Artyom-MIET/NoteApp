# 📝 NotesApp (Java Swing)

Простое десктопное приложение для хранения заметок.

## 🛠 Технологический стек
- Java SE (Red Hat Build of OpenJDK)
- Swing / AWT для графического интерфейса
- java.nio.file для работы с файловой системой
- Без внешних зависимостей

## 🚀 Как запустить
1. Убедитесь, что установлен JDK 11+
2. Откройте папку проекта в VS Code
3. Запустите файл `NoteAppGUI.java` через кнопку "Run" или терминал:
   ```bash
   javac -d out src/NoteAppGUI.java
